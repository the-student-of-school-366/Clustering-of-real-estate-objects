### Кластеризация объектов недвижимости (Unsupervised Learning)
**Технологии**: Python, scikit-learn, K-Means, Silhouette Score

- Применил кластеризацию K-Means для сегментации объектов недвижимости
- Определил оптимальное количество кластеров методом локтя и коэффициентом силуэта
- Реализовал алгоритм K-Means с нуля, включая инициализацию центроидов, назначение кластеров и пересчет центров
- Выполнил анализ кластеров: интерпретация средних значений признаков по кластерам
- Провел сравнение результатов с реализацией из scikit-learn (ARI = 0.88)
